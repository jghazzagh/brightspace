How to embed this content into Brightspace

Follow these steps to seamlessly integrate your interactive layout into your Brightspace course module:

  1. Configure your layout, colors, and interactives using the left-hand panel.
  2. Verify everything looks correct in the Live Preview tab.
  3. Click on the Code Output tab and click the Copy to Clipboard button.
  4. Log in to your D2L Brightspace instance and navigate to your course.
  5. Go to Content and select the module where you want to add the page.
  6. Click New (or Upload / Create) and select Create a File.
  7. Enter a title for your document.
  8. In the HTML Editor toolbar, find and click the Source Code icon (usually represented by </> or similar, at the bottom right or top toolbar depending on your editor version).
  9. Delete any existing code in the Source Code window, then paste the code you copied from this builder.
  10. Click Save in the Source Code window.
  11. Click Save and Close on the main Create a File screen.

Note: Because all CSS is scoped to the .d2l-bldr-wrapper class and JS is encapsulated, this code will not interfere with Daylight styling or other course files!
