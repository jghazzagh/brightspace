Integrating with D2L Brightspace

This layout generator is specifically engineered to bypass Brightspace's HTML sanitizers. It uses inline CSS for colors/fonts and scoped classes for structural grids, ensuring it renders perfectly on both desktop and the Pulse mobile app.

Step-by-Step Guide

  1. Select your colors, typography, and desired layout from the left panel.
  2. Ensure your Contrast Checker shows green "PASS" badges for accessibility compliance.
  3. Verify the design in the Live Preview tab.
  4. Switch to the Code Output tab and click Copy to Clipboard.
  5. Log in to Brightspace, navigate to your Course, and go to Content.
  6. Click Upload / Create and select Create a File.
  7. Enter a Title for your page.
  8. In the HTML Editor toolbar, locate the Source Code icon (usually </>).
  9. Paste the copied code directly into the source code window and click Save.
  10. Click Save and Close. You can now edit the placeholder text directly in the standard Brightspace visual editor!
